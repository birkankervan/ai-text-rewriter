Setup & Installation

Get up and running in less than 2 minutes.

1. Installation (Manual / ZIP)

  • Download Extension Download the latest release .zip file and extract it to a folder.
  • Open Extensions Page In Chrome, go to chrome://extensions.
  • Enable Developer Mode Toggle the switch in the top right corner.
  • Load the Folder Click "Load unpacked" and select the extracted folder, or simply drag and drop the folder onto the page.

2. AI Provider Configuration

You need an API key to use the extension. Choose one below:

Final Step: Add Key to Extension

Click the extension icon, go to Options (Gear Icon), paste your key under the correct provider, and click Save.
